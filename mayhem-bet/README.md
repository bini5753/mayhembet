# ⚔️ 증강 칼바람 딜량 내기 정산기

롤 증강 칼바람(아수라장) 모드에서 딜량 내기를 자동으로 관리하는 로컬 웹 앱.
LCU API로 게임 결과를 자동 수집하고, 순위별 정산 금액을 계산합니다.

---

## 🚀 설치 & 실행

### 1. Python 설치
Python 3.9 이상이 필요합니다. [python.org](https://www.python.org/downloads/)에서 다운로드.
설치 시 **"Add Python to PATH"** 체크 필수!

### 2. 의존성 설치
```bash
cd mayhem-bet
pip install -r requirements.txt
```

### 3. 실행
```bash
python app.py
```

### 4. 브라우저에서 접속
```
http://localhost:5000
```

---

## 📖 사용법

### 1️⃣ 내기 만들기
- 참여 인원 (2~5명) 선택
- 각 참여자의 **소환사명** 입력 (롤 클라이언트에 표시되는 이름 그대로)
- 순위별 정산 금액 설정
  - 예) 3명일 때: 3등→2등에게 1000원, 2등→1등에게 1000원

### 2️⃣ 게임 진행
- 롤 클라이언트를 켜고 증강 칼바람 플레이
- 게임이 끝나면 **자동으로 딜량 순위가 기록**됩니다
- LCU 연결이 안 될 경우 **수동 입력**도 가능

### 3️⃣ 정산
- "정산" 탭에서 실시간 누적 결과 확인
- "내기 종료" 버튼으로 최종 정산 → 누가 누구한테 얼마 내는지 표시

---

## ⚙️ 롤 클라이언트 연결

앱이 자동으로 롤 클라이언트를 감지합니다.
만약 감지가 안 되면:

1. 롤 클라이언트가 실행 중인지 확인
2. 롤 설치 경로가 기본 경로가 아닌 경우 `lcu.py`의 `DEFAULT_LOCKFILE_PATHS`에 경로 추가

**참여자 중 1명만 이 앱을 실행하면 됩니다.** (같은 게임의 모든 플레이어 딜량을 볼 수 있음)

---

## 🔧 문제 해결

| 문제 | 해결 |
|------|------|
| "롤 클라이언트 미연결" | 롤 클라이언트 실행 후 잠시 대기 (5초마다 자동 재연결) |
| 게임 끝났는데 기록 안 됨 | 소환사명이 정확한지 확인 (대소문자 무관) |
| 포트 5000 사용중 | `app.py`에서 `port=5000`을 다른 번호로 변경 |
